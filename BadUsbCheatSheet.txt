BASIC COMMANDS
DELAY <ms> – Pause script for <ms> milliseconds
Works: Windows 10/11, macOS, Linux, iOS, Android

STRING <text> – Type exact text
Works: Windows 10/11, macOS, Linux, iOS, Android

ENTER – Press Enter/Return
Works: Windows 10/11, macOS, Linux, iOS, Android

TAB – Move cursor to next field
Works: Windows 10/11, macOS, Linux, iOS, Android

SPACE – Press Spacebar
Works: Windows 10/11, macOS, Linux, iOS, Android

BACKSPACE – Delete character to the left
Works: Windows 10/11, macOS, Linux, iOS, Android

DELETE – Delete selected item/text
Works: Windows 10/11, macOS, Linux, Android
Doesnt work: iOS

ESCAPE – Cancel action / close dialog
Works: Windows 10/11, macOS, Linux, iOS, Android

MODIFIER KEYS
CTRL – Control key for shortcuts
Works: Windows 10/11, Linux
Doesnt work: macOS (use COMMAND), iOS, Android

ALT – Alt key for menus / shortcuts
Works: Windows 10/11, Linux
Doesnt work: macOS (use OPTION), iOS, Android

SHIFT – Uppercase letters / combined shortcuts
Works: All platforms

GUI – Windows key / Command key / Android launcher key
Works: Windows 10/11, macOS, Linux, Android
Doesnt work: iOS

UPARROW / DOWNARROW / LEFTARROW / RIGHTARROW – Move cursor

HOME – Move cursor to start of line/document

Doesnt work: iOS

END – Move cursor to end of line/document
Works: Windows 10/11, macOS, Linux, Android
Doesnt work: iOS

PAGEUP / PAGEDOWN – Scroll up/down
Works: Windows 10/11, macOS, Linux, Android
Doesnt work: iOS

FUNCTION KEYS
F1-F12 – Standard function keys
Works: Windows 10/11, macOS, Linux
Doesnt work: iOS, Android

ANDROID APP NAVIGATION & INSTALLATION
GUI r → Open launcher/app drawer
TAB / UPARROW / DOWNARROW / LEFTARROW / RIGHTARROW → Navigate app icons
ENTER → Open selected app
REPEAT <n> → Repeat navigation to reach specific app

Educational app install simulation:
GUI r → Open launcher
STRING “Play Store” → Type app store name
ENTER → Open Play Store
STRING <app name> → Search for app
TAB / DOWNARROW → Navigate search results
ENTER → Select app
TAB → Move to Install button
ENTER → Simulate installation

iOS APP NAVIGATION & INSTALLATION
Arrow keys → Navigate app icons on external keyboard
ENTER → Open selected app
STRING <app name> → Search in App Store
TAB / Arrow keys → Navigate search results
ENTER → Select app
TAB → Move to “Get” / “Install” button
ENTER → Simulate installation
Note: Full installation often requires touch input

MOBILE NAVIGATION SHORTCUTS
Android: GUI b → Open default browser
Android: GUI r → Open launcher
Android: ALT TAB → Switch apps (Depends on version)
Android: CTRL c / CTRL v → Copy/Paste in text fields (Depends on keyboard)
iOS: Arrow keys → Navigate text/input fields
iOS: ENTER → Confirm selection
iOS: CTRL c / CTRL v → Copy/Paste (Depends on external keyboard)
